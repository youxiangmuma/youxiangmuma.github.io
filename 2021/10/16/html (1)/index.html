标签

  • 段落——p、span
  • 盒子——div
  • 链接——a
    • href属性为链接地址，
    • target属性为打开方式：
      • _self：当前窗口打开
      • _blank：新窗口打开
  • 图像——img
    • src属性为图片路径
    • title属性为鼠标放置时显示的文本
  • 换行——br
  • 水平线——hr
  • 列表
    • 无序列表——ul-li
    • 有序列表——ol-li
    • 定义列表——dl-dt
  • 表格——table-tr-th
  • 表单——form
    • 属性 ：
      • readonly只读
      • disabled禁用
      • autocomplete自动填充on、off
      • autofocus自动获取焦点
      • placeholder提示
      • required必填字段
  • 下拉列表——select-option
  • 列表——datalist-option
  • 多行文本框——textarea
  • 音频——audio

css注释
选择器

  • id选择器：.

  • 类选择器：#

  • 通配选择器：*

  • 复合选择器：同时满足多个条件，各选择器用空格分开

  • 选择器分组：同时多个选择器，各选择器用逗号分开

  • 后代选择器：指定元素的后代，祖先元素 后代元素

  • 子元素选择器：指定父元素的子元素，父元素>子元素

  • 兄弟选择器：

    • 紧挨的一个兄弟元素：前一个+后一个
    • 后面所有的兄弟元素：兄~弟

  • 伪类选择器：

    • :first-child 第一个子元素

    • :last-child 最后一个子元素

    • :nth-child() 选中第n个子元素

      特殊值：

      • n 第n个 n的范围0到正无穷

      • 2n 或 even 表示选中偶数位的元素

      • 2n+1 或 odd 表示选中奇数位的元素

  • a标签的伪类

    • a:link{声明块}——未访问过的链接

    • a:visited{声明块}——访问过的链接

    • a:hover{声明块}——鼠标滑过的链接

    • a:active{声明块}——正在点击的链接

  • 伪元素选择器：

    • ::first-letter 表示第一个字母

    • ::first-line 表示第一行

    • ::selection 表示选中的内容

    • ::before 元素的开始

    • ::after 元素的最后

      • before 和 after 必须结合content属性来使用

      • ```html

            div::before{
        content: '『';
     }

    div::after{
        content: '』';
    }
元素属性
  • 修改元素类型 dispaly
    • inline将元素作内联元素；
    • block将元素作块元素
    • inline-block将元素作行内块元素（既有行内元素的特点又有块元素的特点）
    • none隐藏元素——不占页面位置
  • 内联元素块元素区别

  • 设置元素状态 visibility

    • visible显示；hidden不显示——占页面位置

  • 父元素处理溢出问题 overflow

    • visible显示在父元素之外；
    • hidden去除溢出内容；
    • scroll为父元素添加滚动条：scroll-x水平滚动条；scroll-y垂直滚动条
    • auto根据需求添加滚动条
浮动 float

  • none默认值；left左侧浮动；right右侧浮动

  • 当元素设置浮动以后，会立即脱离文档流，遇到父元素或其他浮动元素会停止，不会超过父元素或其他浮动元素

    如果浮动元素上是一个没有浮动的元素，则浮动元素也不会超过块元素，浮动元素也不会超过上边的浮动元素

    浮动的元素不会盖住文字，文字会自动环绕

  • 块元素脱离文档流，宽度和高度都会被内容撑开

    内联元素脱离文档流，会变成块元素，宽度和高度都可以设置，默认被内容撑开————即块被内容撑开，内联元素变成块元素

  • 高度塌陷

  • https://blog.csdn.net/tiankai100/article/details/83054061——网址

  • 清除浮动 clear

    • none不清除；left清除左侧；right清除右侧；both清除两侧
定位

  • 设置元素定位 position

    • static不开启定位
    • relative相对定位
      • 相对于元素在文档流中原来的位置进行定位，不改变元素性质
    • absolute绝对定位
      • 相对于离他最近的开启了定位的祖先元素进行定位，改变元素性质——块变行内
    • fixed固定定位
      • 固定定位永远都是相对于浏览器窗口定位

  • 偏移量

    • left、right、top、bottom——左边距、右边距、上边距、下边距

框架

  • rows——指定框架集中所有框架，一行一行排列
  • cols——指定框架集中所有框架，一列一列排列

  • iframe ——body里写
